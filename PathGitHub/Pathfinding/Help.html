Help

Place start

When you want to place the start of your pathfinding you click on start and place your start in the grid

Place wall

The wall blocks the path and the path is forcd to take a route around it, you can place walls by click on the butten and press in the grid

Place end

When you want to place the ond of your pathfinding, click on end and place the end

Eraser

If you missplaces an wall or your start or end you can use the eraser to clear the wall, start or end

Save

If you want to save your walls and your start and end you can save here

Load

if you want to load an older set of walls you can load here, but you need the same gride size as before

Calculate

If you want to calculate the path from start to end you click on calculate

Keybinds

  • If you push 1 you will start the pathfinding.
  • If you push 2 you will clear the grid.
  • If you push 3 you will clear the path but not the walls.
